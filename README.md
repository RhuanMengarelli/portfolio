## 🐳 Development Environment

This project was developed inside an isolated development environment using **Docker** and **VS Code Dev Containers**.

This approach guarantees a consistent environment across different machines without requiring Node.js or project dependencies to be installed directly on the host operating system.

---

## 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/RhuanMengarelli/portfolio.git
```

Open the project in **Visual Studio Code**.

If you use **Dev Containers**, reopen the project inside the container.

After that, start the project using **Live Server** or any static web server of your choice.
